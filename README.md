## 📌 Sobre o repositório

Este repositório contém uma coleção de **exercícios e mini projetos em JavaScript**, feitos com o objetivo de melhorar a lógica de programação e dominar o desenvolvimento web.

---

## 🎯 Objetivos

- 💡 Praticar lógica de programação
- 🧠 Melhorar habilidades em JavaScript
- 🛠️ Criar mini projetos reais
- 📚 Aprender manipulação do DOM
- 💾 Trabalhar com LocalStorage
- 🚀 Evoluir como desenvolvedor frontend
